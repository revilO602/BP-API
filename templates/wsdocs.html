Websocket service for tracking courier positions.

Servers

  • wss://poslito.com/wswsspublic

Operations

  • PUB /couriers

    Send current position to all subscribers. Only authenticated courier can publish.

    string

    Query parameters e.g. /?token=someuuid

    token
    string

    JWT access token of the courier. Required to send position over the websocket.

    Accepts the following message:

    CourierPosition

    Geographical coordinates of the courier.

    object
    uid: courierPosition
    latitude
    number
    format: float[ -90 .. 90 ]
    longitude
    number
    format: float[ -180 .. 180 ]

    Additional properties are allowed.

    Examples

    • #1 Example
      {
  "latitude": 50.6548,
  "longitude": 62.2415
}
